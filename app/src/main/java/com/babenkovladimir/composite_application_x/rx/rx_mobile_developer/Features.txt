Есть 3 основных источника данных.
Observable
Flovable
Single

Observable - основной класс. Всего 5 классов. Flowable имеет все теже самы возможности.

А Вот Single - имеет только onSuccess и onError - для какойто единичной операции - это типа как для отдельного колбэка
Кдасс мейби - onSuccess, onError, OnCompleate - 
Compleatable - либо завершился либо нет


Flowable - это backpressure. Соответсвенно когда поток данных менее 1000 данных в секунду. Гипотетически - сокеты
Compleatable - главное знать, выполнилось или нет. Например свичнуть какуюто радиобаттон
Maybe - если есть результат - отобразить. (типа трай кетч или чтото типа того
